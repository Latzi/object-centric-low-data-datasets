# Object-Centric Low-Data Datasets

This repository contains the release assets for a curated collection of three object-centric datasets used for low-data image generation and augmentation.

## Subsets
- Traffic Signs
- Cityscapes--Pedestrian
- COCO PottedPlant

## Release model
- Traffic Signs: direct release
- Cityscapes--Pedestrian: non-image / abstract release
- COCO PottedPlant: metadata / reconstruction-first release

See `docs/release_matrix.md` for details.
